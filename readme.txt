מדריך הפעלה לפרויקט

מה יש בפרויקט:
server (.NET WebAPI)
צד שרת לניהול מוסכים. מבצע שליפה של מוסכים מה-API הממשלתי, שמירה במסד הנתונים והוספה של מוסכים חדשים. כל פעולות ה-DB אסינכרוניות.

נקודות קצה:
GET /api/garage/fetchFro
GET /api/garage/saved – שליפת מוסכים שמורים במסד
POST /api/garage/add – הוספת מוסכים חדשים למסד


client (Angular)
דף ראשי המציג טבלה עם מוסכים שמורים ו-Multi Select לבחירת מוסכים חדשים להוספה. השירות ב-Angular מתקשר עם נקודות הקצה שבשרת.

הפעלה והרצה:
צד שרת:
פתחו את פרויקט ה-API ב-Visual Studio
ודאו שה-.NET SDK מותקן ומוגדר כראוי
הריצו את הפרויקט (F5 או Run)
השרת יאזין בכתובת מקומית

צד לקוח:
פתחו את תיקיית הפרויקט ב-Visual Studio Code או IDE אחר התומך ב-Angular
התקינו את התלויות: npm install
הריצו את הפרויקט: ng serve
הדפדפן יפתח אוטומטית בכתובת מקומית

שימוש:
בעת טעינת הדף הראשי, המוסכים מה-API הממשלתי נטענים ל-Multi Select
ניתן לבחור מוסכים ולהוסיף אותם למסד הנתונים
מוסכים שנוספו יוצגו בטבלה
כל פעולה אסינכרונית (טעינה או הוספה) מציגה חיווי למשתמש
